Deploy Terraform Infra
-Write
-Plan
-Apply

terraform init
-prepare configuration for use
-find required provider plug-ins
-configure state backend
  Uses working directory by default
-Rerun ONLY when state backend, providers, or modules change

terraform plan
-loads configuration from working directory
-loads state data into memory
-compares configuration and state
-creates an execution plan
  save with -out option  e.g. terraform plan -out somefile.tfplan
  Use terraform show <plan file> to see its contents

+ resource will be created
- resource will be destroyed
~ resource will be modified in place
-/+ resource will be recreated

terraform apply
-accepts a saved plan file
-executes changes from plan
-updates state data contents
-generates a plan if none submitted

terraform destroy
-destroys all resources from state
-use with caution
-alias for terraform apply -destroy

Terraform Data Types
-Primitive - String, number, bool
-Collection - List, set, map
-Structural - Tuple, object

Data Type Examples
# List
[1, 2, 3, 4]

#Map
{
    small = "t2.micro"
    medium = "t2.small"
    large = "t2.large"
}

#Collections
variable "aws_regions" {
    type = list(string)
    description = "Regions for AWS resources"
    default = ["us-east-1", "us-east-2", "us-west-1", "us-west-2"]
}

Reference Collection Values
var.<name_label>[<element_number>] - 0 based start value
var.aws_regions[0]

#Map
variable "aws_instance_sizes" {
    type = map(string)
    description = "Instance sizes to use in AWS"
    default = {
        small = "t3.micro"
        medium = "m4.large"
        large = "m4.xlarge"
    }
}

Reference Collection Values
var.<name_label>.<key_name> or var.<name_label>["key_name"]
var.aws_instance_sizes.small or var.aws_instance_sizes["small"]

Data Encryption
Access Control

terraform {
    backend "backend_type" {
        IDENTIFIER = LITERAL_VALUE
    }
}

e.g.
terraform {
    backend "s3" {
        bucket = "globo-state-12345"
        key = "dev/terraform.tfstate"
        #these can be removed below
        #region = "us-east-1"
        #access_key = "xxxxxx"
        #secret_key = "yyyyyy"
    }
}

Moved Block - moved.tf move state
moved {
    from = ADDRESS
    to = ADDRESS
}

Import Block - imports.tf 
import {
    to = ADDRESS
    id = "UNIQUE_IDENTIFIER"
}

Removed Block - removed.tf
removed {
    from = ADDRESS
    lifecycle {
        prevent_destroy = false
    }
}

Migrate State Data
-Add backend config
-Run terraform init -backend-config="key=dev.tfstate" (name of file to create)
-Answer yes to migrate existing state data
-Run a terraform plan - should show no changes meaning showing it has moved to your backend
